Explain why the result of ('b' + 'a' + + 'a' + 'a').toLowerCase() is banana.

  • 'b' + 'a' results in "ba" because the + operator is used for string concatenation.
  • + 'a' attempts to convert the string "a" into a number. Since it's not a valid number, it evaluates to NaN (Not-a-Number).
  • ('ba' + NaN) combines the string "ba" with NaN, which results in the string "baNaN".
  • .toLowerCase() is called on the string "baNaN", which converts all characters to lowercase.
